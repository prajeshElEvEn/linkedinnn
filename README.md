# LinkedIn Clone

`4` hour project to build a LinkedIn clone using `React`, `Redux`, and `Firebase`.

## Features

- Firebase Authentication
- Post feed
- Clean UI
- Deployed to Firebase

- Run `npm install` to install dependencies
- Run `npm start` to start the app
